# FluCash - Gerenciador Inteligente de Gastos

[![FluCash CI](https://github.com/SEU_USUARIO/flucash/actions/workflows/ci.yml/badge.svg)](https://github.com/SEU_USUARIO/flucash/actions/workflows/ci.yml)

## O Problema
Muitas pessoas perdem o controle de suas finanças devido aos pequenos gastos do dia a dia. A falta de registro rápido e simples gera ansiedade financeira e endividamento.

## A Solução
O FluCash é uma aplicação CLI leve que permite registrar despesas diárias, categorizá-las e acompanhar resumos detalhados para uma melhor educação financeira.

## Funcionalidades
* Cadastro de gastos com valor, descrição e Categoria.
* Validação automática de dados incorretos.
* Cálculo do total geral acumulado.
* Resumo Financeiro Avançado: Visualização de gastos agrupados por categoria.

## Tecnologias Utilizadas
* Java 17
* Maven
* JUnit 5
* Checkstyle
* GitHub Actions
